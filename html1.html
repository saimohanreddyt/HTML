Linux Commands

Kali_Linux commands helps to the linux users

  1. date -Display the current time in the certain FORMAT, or set the system date.
  2. cal -Display the current date in the calender format
  3. uptime -show uptime in pretty format
  4. whoami -Print the user name associated with the current effective user ID.
  5. uname -Print certain system information
  6. users -Output who is currently logged in according to FILE.
  7. free -show detailed about memory usage and rest other free space.
  8. ps -Shows the baground running processes
  9. cd -Change the current directory to DIR. The default DIR is the value of the HOME shell variable.
  10. pwd -Print the name of the current working directory.
  11. mkdir -Create the DIRECTORY(ies), if they do not already exist.
  12. ls -List information about the FILEs (the current directory by default).
  13. cat -Concatenate FILE(s) to standard output.
  14. touch -Update the access and modification times of each FILE to the current time.
  15. echo -echo is a command used to print content what we entered along with echo(echo "GOOD MORNING")
  16. nano -To place the cursor on a specific line of a file, put the line number with a '+' before the filename. The column number can be added after a comma. When a filename is '-', nano reads data from standard input.
  17. grep -Search for PATTERNS in each FILE.
  18. cp -Copy SOURCE to DEST, or multiple SOURCE(s) to DIRECTORY.
  19. mv -Rename SOURCE to DEST, or move SOURCE(s) to DIRECTORY.
  20. rm -Remove (unlink) the FILE(s).
  21. less -
  22. mor -A file perusal filter for CRT viewing.
  23. sort -Write sorted concatenation of all FILE(s) to standard output.
  24. chown -Change the owner and/or group of each FILE to OWNER and/or GROUP.
  25. chmod -Change the mode of each FILE to MODE.
  26. vi/v -
  27. history -Display the history list with line numbers, prefixing each modified entry with a `*'. An argument of N lists only the last N entries.
  28. -v -
  29. man + help -Mandatory or optional arguments to long options are also mandatory or optional for any corresponding short options.
Listing Directory Content
  • -l long listing
  • -a list hidden files
  • -r list in reverse name
  • -t list new files first
  • -rt list in reverse time(old order)
